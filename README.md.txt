# Console Calculator

This is a simple console-based calculator program written in Java. 
It performs basic arithmetic operations: addition, subtraction, multiplication, and division. 
The results of the calculations are logged to a file named `calclog.txt`.

# Features

- Addition: Adds two numbers.
- Subtraction: Subtracts the second number from the first.
- Multiplication: Multiplies two numbers.
- Division: Divides the first number by the second (with error handling for division by zero).
- Logging: All operations and their results are logged to a file.
